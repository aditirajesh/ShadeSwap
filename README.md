# ShadeSwap
ShadeSwap is a fun and engaging application designed to apply a sunglasses filter to images. Utilizing advanced image processing techniques, ShadeSwap allows users to seamlessly add stylish sunglasses to photos, enhancing them with a touch of flair and personality. 

## Key Features:

1. Sunglasses Filter: Effortlessly apply a variety of sunglasses styles to any image.
User-Friendly Interface: Intuitive design for easy use and quick editing.
Customization Options: Adjust the size, position, and style of sunglasses to fit the image perfectly.
Technologies Used:

2. OpenCV: For image processing and manipulation.
3. Python: Backend scripting for filter application and image handling.
